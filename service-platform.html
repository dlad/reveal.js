Service Platform

November 2013

What is it?

Many different things...

  • Web Site
  • Web Server
  • Application Server
It means different things to different people: end users (web site), developers (java application). This presentation is about the application called the service-platform

How is it built?

Java, Jetty, Maven, Chef

What does it talk to?

  • Databases – clouddb, myclouddb hsppdb
  • NAC – session management
  • Omniture – web analytics
  • SMS Gateway
  • Mail Server

What talks to it?

  • Apache
  • Monitoring
High level – end user devices, but actually very few things talk directly to it on a low level...
