Configuration

  • Application Configuration
  • Platform Configuration

Application config is read locally; unique to instance.

Platform configuration is read from the database and is localized to the end user

Application Configuration

  • application.properties
  • jetty.xml
  • logback.xml

application properties and jetty xml require restart. logback xml can be changed on the fly (watch out for chef!)

Platform configuration is read from the database and is localized to the end user – two parts: defining a configuration; and assigning it to a location

Platform Configuration

  • Stored in the database
  • configuration.*
  • shared.feature_configuration

Cached, TTL 30 min, usually done as BAU changes, managed by product, applied by DBA
