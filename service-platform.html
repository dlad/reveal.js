What is it?

Many different things...

  • Web Site

This presentation is about the application called the service-platform which encompassed all of these.

What talks to it?

Configuration

  • Application Configuration
  • Platform Configuration

Application config is read locally: appplication.properties, jetty.xml, logback.xml – unique to instance.

Platform configuration is read from the database and is localized to the end user – two parts: defining a configuration; and assigning it to a location
